Instalación rápida

El sistema trabaja con Google Sheets como base de datos y Apps Script como API.

  1. Sube o crea una hoja de Google Sheets.
  2. Abre Extensiones > Apps Script.
  3. Pega el archivo appscript/Code.gs.
  4. Ejecuta setupDatabase() una vez para crear las hojas y cargar perfumes.
  5. Implementa como Aplicación web con acceso “cualquier usuario con el enlace”.
  6. Copia la URL terminada en /exec y reemplaza la constante API_URL en este HTML.
  7. Sube la carpeta frontend a Vercel, Netlify o cualquier hosting estático.
Por seguridad, este archivo no trae tu URL real de Apps Script porque debe generarse desde tu cuenta de Google. La conexión queda interna en el código apenas pegues esa URL.
